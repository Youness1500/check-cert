# check-cert

🔐 Certificate Checker Utility

## Installation

```bash
make install
```

## Usage

```bash
check-cert -c mycert.pem -k mykey.pem
check-cert -c mycert.pem -k mykey.pem -i intermediate.pem -v
```

## Manual

```bash
man check-cert
```


## Uninstall 
```bash
make uninstall
```
